# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its code of conduct.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Summary

We are committed to providing a welcoming and inclusive environment for
everyone. All participants — contributors, maintainers, and users — are
expected to treat each other with respect.

Unacceptable behavior will not be tolerated. Maintainers may remove, edit,
or reject contributions that do not align with this code of conduct, and may
ban contributors whose behavior is deemed inappropriate.

## Reporting

If you experience or witness a violation, please report it privately to the
project maintainer:

**Jonathan Castro Miguel** — jonathan.castro.miguel@gmail.com

All reports will be handled with discretion.

## Scope

This code of conduct applies within all project spaces (repository, issues,
pull requests, discussions) and in any public space where an individual is
representing the project.
